Case Study
by Bi-Silque


Requisitos:
A tua presença.


+ Info

A Bi-Silque, fundada em 1979, é uma empresa criada por um casal empreendedor numa garagem em Esmoriz. A paixão por criar aplicações novas e inovadoras com cortiça, tanto para a casa como para o escritório, transformou este pequeno negócio numa empresa multinacional que continua em constante crescimento.

Esta ideia criativa, pensada há 40 anos, resultou numa fusão única de design e funcionalidade que é evidente em todos os produtos que esta empresa fabrica hoje em dia.

Líder global em fabrico e distribuição de produtos de comunicação visual, a Bi-Silque está sediada em Portugal onde fabrica e exporta uma ampla seleção de produtos sob as marcas Bi-Office, Bi-Bright e MasterVision para mais de 80 países em 5 continentes.

Para além de se focar na inovação, têm como objetivo ser líder também na sustentabilidade e é por essa razão que vão estar presentes.

Introdução a Web
Components


Requisitos:
• Conta GitHub
• Conhecimentos básicos HTML, CSS e JS
• Computador com browser (Chrome ou Firefox)


+ Info

Nos dias de hoje, com a constante e rápida evolução das tecnologias utilizadas em desenvolvimento Frontend sente-se cada vez mais que estamos constantemente a reinventar a roda. Estamos a resolver os mesmo problema de acordo com o use-case, projecto ou mesmo as tecnologias de desenvolvimento. Como podemos minimizar este problema e poupar tempo?

Nesta introdução a Web Components vamos:

1. Identificar os problemas existentes no desenvolvimento Frontend na utilização de frameworks

2. Perceber o que são Web Components e qual o seu objectivo

3. Identificar as vantagens e desvantagens de os utilizar alguns dos use-cases

4. Criar componentes genéricos que podem ser reutilizados entre projetos, mesmo que usem tecnologias diferentes
